# tft-discord-bot
A Discord bot that automatically tracks TFT match results using Riot API.
TFT Discord Bot
This project is a Discord bot that tracks Teamfight Tactics (TFT) match history using the Riot Games API.

It automatically detects when a match has ended and sends match results (placement, level, damage dealt, etc.) to a Discord channel.

Features
Track TFT match history using Riot API
Detect new completed matches automatically
Send results to Discord

Purpose
This project is for personal use and a small group of friends to track TFT performance and improve gameplay.

Tech Stack
Python
Discord.py
Riot Games API
